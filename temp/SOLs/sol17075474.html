sol17075474: glibc vulnerability CVE-2016-4429
Security Advisory

Original Publication Date: 07/27/2016

Vulnerability Description

Stack-based buffer overflow in the clntudp_call function in sunrpc/clnt_udp.c in the GNU C Library (aka glibc or libc6) allows remote servers to cause a denial of service (crash) or possibly unspecified other impact via a flood of crafted ICMP and UDP packets. (CVE-2016-4429)

Impact

The vulnerable code is present on specified F5 systems, however F5 products are not vulnerable in recommended or default configurations, and there is no known impact at this time.

Security Issue Status

F5 Product Development has assigned ID 602290 (BIG-IP), ID 602537 (BIG-IQ and F5 iWorkflow), ID 602540 (Enterprise Manager), and ID 476571 (ARX) to this vulnerability.
